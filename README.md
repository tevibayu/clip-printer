## CLIP PRINTER
Easy direct file print from Server to Client Browser (POS/ESC Printer or Universal Printer)

## Features

- Powerfull
- Fast Direct
- Easy
- Portable
- Supported : .zpl, .pdf, .txt, .jpg, .png

## How it work

Install Clip Printer application desktop and just include file javascript to your script.
